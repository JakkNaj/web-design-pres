# Navrh02

Tato složka je čistý obsahový základ pro nový návrh webu Česká dopravní stavba.

- Původní vizuální návrh a všechny jeho komponenty jsou v `../navrh01`.
- Obsahové JSON kolekce jsou zachované v `src/content`.
- Základní brand data jsou v `src/data/brand.ts`.
- Logo a favicon stažené z aktuálního webu jsou v `public/brand`.
- Globální CSS obsahuje jen brand proměnné a minimální čitelnost, žádné navržené sekce.
- Stránky v `src/pages` slouží jako placeholder výpisy obsahu pro další designovou práci.
